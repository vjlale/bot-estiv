# Fuentes de marca — Gardens Wood

Fuentes oficiales de Gardens Wood según el manual. Ambas libres bajo
SIL Open Font License 1.1.

## Archivos

| Archivo | Uso | Fuente original |
|---|---|---|
| `PlayfairDisplay-Regular.ttf` | titulares elegantes (variable font) | Google Fonts |
| `PlayfairDisplay-Bold.ttf` | titulares peso alto | Google Fonts (gwfh latin subset) |
| `Montserrat-Regular.ttf` | cuerpo / captions | github.com/JulietaUla/Montserrat |
| `Montserrat-Medium.ttf` | subtítulos | github.com/JulietaUla/Montserrat |
| `Montserrat-SemiBold.ttf` | labels / pilar tags / CTAs | github.com/JulietaUla/Montserrat |
| `Montserrat-Bold.ttf` | énfasis fuertes | github.com/JulietaUla/Montserrat |

Estas fuentes se consumen desde Pillow vía
`bot_estiv.tools.canvas_design._load_font` usando la ruta
`packages/brand/fonts/` resuelta relativa al root del monorepo.
