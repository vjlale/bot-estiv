# Bot Estiv — Brand package

Tokens y assets compartidos entre el backend Python (`bot_estiv.brand`) y el
dashboard TypeScript (`tailwind.config.ts`).

## Fuentes de verdad

- `../Branding/Manual de Identidad de Marca_ Gardens Wood.txt` → texto base
- `../Branding/LOGOCOMPLETO.png` → logo principal
- `apps/api/src/bot_estiv/brand.py` → tokens Python
- `apps/dashboard/tailwind.config.ts` → tokens CSS

Cualquier cambio de marca debe reflejarse en los 3 lugares.
